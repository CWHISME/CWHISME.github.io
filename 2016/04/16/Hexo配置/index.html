# 执行命令

npm install-g hexo

(可选，若已提交至 Git)：

hexo init

Hexo 3.0 以上版本，需单独设置此项才可自动进行 Deploy：

npm install hexo-deployer-git --save

_config.yml 配置：

deploy:
  type: git
  repo: 
     coding: https://[项目地址],[项目分支]
