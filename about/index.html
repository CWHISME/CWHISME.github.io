# CWHISME

# 简介

  • 一个平凡而又普通的程序员

    • 给我网和食物，可以宅到世界末日... 😶‍🌫️

  • 梦想做出一个自己能喜欢的游戏

    • 比如令我印象深刻的
      • 《吞食天地》—— 当年玩的第一个中文 RPG 游戏，玩过好多遍
      • 《勇者斗恶龙 IV》—— 分章节的剧情，商人章节的宝箱掉落概率好高，刚开始草药就经常掉
      • 《勇者斗恶龙 III》—— 神殿转职系统、贤者。也是在这上面玩了很久，记得玩的是外星科技的卡带，刚才查了一下那会叫 DG6，那就是这个了
      • 《最终幻想 2》—— 前期偶然刷到青蛙术就感觉非常有趣了，令人印象深刻，熟练度系统也是
      • 《封神榜》—— 连击忘了是什么原理了，速度？
      • 《重装机兵》—— 记得铁弹能直接打坏机械敌人的设备、战车改造很多样，还有.. 红狼、诺亚
      • 《勇者斗恶龙 V》—— 是后面玩的了，芙萝拉、比安卡... 太久了，刚才查了下才又听到说还可以有怪物同伴？
      • 《勇者斗恶龙怪兽篇》—— 当年好像是在还没有智能手机的时代，GBC 版本吧应该是？用 Java 模拟器玩的，记得剧情是从船上醒来，去一个岛，怪物合成、玩法真是的好丰富
      • 《口袋精灵》—— 差点忘了这个，当年也是玩过几代的，玩完了还找小说看
    • 都是 NES 版本的
    • 其次是大学时候的《上古卷轴 5》，不算类型的话《死亡空间 2》
      • 当年就在 I5-3210 核心显卡上玩，现在想来，这优化可真是挺厉害的
