# 总结

  • 无自定义带参数构造函数，结构体直接外部调用 initobj 初始化
  • 带自定义带参数构造函数，结构体自定义构造函数中可调用 new 操作符执行 initobj 初始化
  • 带自定义带参数构造函数，初始化赋值走正常赋值

initobj：将位于指定地址的值类型的每个字段初始化为空引用或适当的基元类型的 0
